📂 KRX 수동 데이터 업로드 주 1~2회 수동 갱신
📋 KRX CSV 다운로드 방법
1. data.krx.co.kr 접속
2. 주식 → 투자자별 거래실적 → 일별 투자자별 매매현황 → 종목별 CSV (수량 기준)
3. 주식 → 종목시세 → 개별종목 시세 → 종목별 CSV
4. 시세 11개 + 투자자 11개 = 22개 파일 한번에 업로드
